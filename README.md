JavaScript Logic & State Management – To-Do List Application

📌 Project Overview

This project is an interactive To-Do List Application built using HTML, CSS, and JavaScript. It demonstrates core JavaScript concepts such as DOM manipulation, event handling, state management, and browser-based data persistence using "localStorage".

The application allows users to create, manage, update, and track their daily tasks efficiently. All task data is automatically saved in the browser and remains available even after page refreshes or browser restarts.

---

🚀 Features

✅ Create Tasks

- Add new tasks instantly through an input field.
- Validate empty task entries.

✏️ Update Tasks

- Edit existing tasks without deleting them.
- Save modifications dynamically.

🗑️ Delete Tasks

- Remove individual tasks from the list.
- Automatically update local storage.

✔️ Mark as Completed

- Toggle task completion status.
- Visually distinguish completed tasks.

🔍 Advanced Filtering

- All Tasks – Displays every task.
- Active Tasks – Shows incomplete tasks only.
- Completed Tasks – Shows finished tasks only.

💾 Local Storage Persistence

- Automatically stores task data using "window.localStorage".
- Restores tasks when the application reloads.

🎯 Dynamic DOM Manipulation

- Dynamically creates task elements.
- Updates UI without page refresh.

⚡ Event Delegation

- Uses delegated event listeners for efficient event handling.
- Improves performance and scalability.
